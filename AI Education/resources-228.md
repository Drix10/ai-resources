### 🤖 6-DoF Pose Tracking - ProxyPose Method

ProxyPose introduces a method for 6-DoF pose tracking that uses a fine-tuned video diffusion model. This model generates proxy motion from a single marked pixel, which then allows for pose recovery with standard vision code.

Key Points:

• ProxyPose uses a single pixel mark to initiate tracking.

• A video diffusion model generates a proxy video showing local motion.

• Standard vision code recovers the 6-DoF pose from the proxy video.

• This approach does not require CAD models or depth data.

🚀 Implementation:
1. Mark a pixel on the object of interest in a single video frame.
2. Use a fine-tuned video diffusion model to generate a proxy video mimicking the local motion.
3. Apply classic vision algorithms to the proxy video to determine the object's 6-DoF pose.
