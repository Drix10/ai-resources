• The 20-year-old platform transitions into a maintenance state.

Key Points:

• Phosh 0.56.0 supports atomic updates for mobile Linux.

• App hiding is implemented for immutable OS images.

### 💡 Cybersecurity - Supply Chain Attack Vector

A Rust maintainer reported a sophisticated supply-chain attack attempt via a fake VC interview. The attackers aimed to establish a foothold on his system. This incident highlights new methods for malware distribution.

Key Points:

• An attacker attempted to compromise a Rust maintainer's laptop.

• The attack used a fake VC interview as a social engineering tactic.

• Malware, PinpinRAT, was hidden within a TypeScript patch and a PNG file.
