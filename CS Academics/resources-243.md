### ✨ GitHub Release Pages - UI Enhancements

GitHub has updated release pages with new features aimed at improving navigation and providing direct access to asset download metrics. These enhancements streamline the process of managing and understanding software releases.

Key Points:

• New sidebar table of contents improves navigation within release pages.

• Download counts for individual assets are now visible directly in the UI.

• Access to download metrics no longer requires using the API for basic insights.
