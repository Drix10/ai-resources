### 🤖 AI Development - Genkit Agents API

This article introduces the Genkit Agents API, designed to simplify the creation of agentic, full-stack AI applications. The API manages common complexities such as message history, streaming, and tool execution loops.

Key Points:
• Simplifies building agentic, full-stack AI applications.

• Addresses plumbing challenges for message history, streaming, and tool loops.

• Enables defining AI agents directly on the server.

• Available in a preview version for TypeScript and Go.
